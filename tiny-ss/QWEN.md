# Spring Security 单体应用项目

## 项目概述

这是一个用于学习和实践 Spring Security 在 Spring Boot 单体应用中用户认证和授权的完整项目。通过本项目，您可以掌握从基础的用户认证到高级安全概念的全面知识。

## 项目目标

### 核心目标
- 实现完整的用户登录、注册和登出功能
- 配置基于数据库的用户管理
- 实现基于角色的访问控制 (RBAC)
- 集成 Thymeleaf 模板引擎的安全标签
- 配置 H2 内存数据库用于开发测试

### 进阶目标
- 深入理解 Spring Security 的过滤器链机制
- 掌握自定义认证流程和权限检查
- 学习密码安全策略（BCrypt 加密）
- 掌握安全配置的最佳实践
- 理解 CSRF 保护和会话管理
- 实现记住我功能（可选）
- 配置安全头（如 X-Frame-Options）

### 附加学习目标
- 了解 OAuth2 和 OpenID Connect 基础概念（为进一步学习做准备）
- 学习安全审计和日志记录
- 掌握安全测试方法
- 了解常见安全漏洞和防护措施（如注入攻击、XSS、CSRF 等）
- 实践安全配置的性能优化
- 掌握安全事件的监控与处理
- 理解安全配置的生产环境部署考虑因素

## 项目特性

- **用户管理**: 完整的用户注册、登录、登出流程
- **角色权限**: 基于角色的访问控制（USER、ADMIN）
- **前端集成**: Thymeleaf 模板与 Spring Security 集成
- **数据库支持**: 使用 JPA 和 H2 数据库存储用户信息
- **安全配置**: 全面的安全配置和保护机制
- **欢迎页面**: 优化用户体验，非默认跳转至登录页

## 学习成果

完成本项目后，您将能够:
- 在 Spring Boot 应用中独立实现完整的安全认证系统
- 配置和管理用户权限体系
- 集成数据库进行用户信息存储和管理
- 配置安全相关的最佳实践
- 理解 Spring Security 的核心概念和内部机制
- 为后续学习微服务安全、OAuth2 等高级主题打下坚实基础
